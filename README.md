    # Chatbot Persona Factory

    A lightweight Python application demonstrating how to use **Object-Oriented Programming (OOP)** and the **Factory Design Pattern** to generate distinct AI personalities using the OpenAI API.

    Choose between Filipino Kabayan, Marites na Kapitbahay, and Girl Bestfriend. Add more and enjoy the humor!

    ## Overview

    This project is built for **learning and practice**. It showcases how to:

    - Encapsulate AI logic using Classes (`Persona` and `ChatBot`).
    - Implement a `BotFactory` to simplify the creation of different chatbot types.
    - Manage conversation state/history.
    - Integrate with OpenAI's `gpt-4o-mini` model.
    - Use Git for version control

    ## Tech Stack

    - **Language:** Python 3.x
    - **AI Engine:** OpenAI API (`gpt-4o-mini`)
    - **Libraries:** `openai`, `python-dotenv`
